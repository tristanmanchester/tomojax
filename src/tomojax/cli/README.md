# tomojax.cli

`tomojax.cli` owns the grouped `tomojax` console script. The installed command exposes only product workflows: `inspect`, `validate`, `preprocess`, `ingest`, `convert`, `recon`, `align`, and `simulate`.

Benchmark and diagnostic dispatch has been removed from the product CLI. One-off runners and historical `tomojax dev ...` commands are archived outside the publishable package.
